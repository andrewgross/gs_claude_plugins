# gs-claude-plugins

Personal collection of Claude Code plugins.

## Usage

Add this marketplace to Claude Code:

```bash
/plugin marketplace add /path/to/gs_claude_plugins
```

Then install any plugin:

```bash
/plugin install <plugin-name>@gs-claude-plugins
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [terminal-notifier](./plugins/terminal-notifier) | Native macOS notifications when Claude finishes responding |
